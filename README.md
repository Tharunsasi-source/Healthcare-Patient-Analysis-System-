1. Project Title

Healthcare Patient Analysis System

2. Introduction

The Healthcare Patient Analysis System is a data-driven project designed to generate, manage, and analyze patient health records. It helps in identifying patterns, detecting health risks, and supporting decision-making in healthcare.

3. Objectives
Generate synthetic patient data using Faker
Analyze patient health metrics
Identify high-risk patients
Visualize healthcare trends
Improve decision-making using data

4. Problem Statement

Hospitals and healthcare institutions generate large amounts of patient data. Manual analysis is time-consuming and error-prone. This system automates data generation and analysis to provide insights efficiently.

5. Technologies Used
Python
Pandas
NumPy
Matplotlib / Seaborn
Faker

6. Dataset Description

The dataset contains 10,000 synthetic patient records with the following attributes:

Column Name	Description
Patient_ID	Unique ID
Age	20–80
Gender	Male/Female
Blood_Pressure	80–180
Sugar_Level	70–200
Cholesterol	150–300
Heart_Rate	60–120

7. Methodology

Data Generation using Faker
Data Preprocessing
Exploratory Data Analysis (EDA)
Visualization of health metrics
Risk classification

8. Features

Synthetic data generation
